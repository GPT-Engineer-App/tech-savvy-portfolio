# tech-savvy-portfolio

A personal website for a programmer, featuring sections for an about page, portfolio of projects, a blog, and contact information. The design will be sleek and professional, with a modern layout and a color scheme that reflects a tech-savvy, innovative personality. The website should be user-friendly, mobile-responsive, and highlight the programmer's skills and past work effectively.

## Setup

```sh
git clone https://github.com/GPT-Engineer-App/tech-savvy-portfolio.git

```


This will run a dev server with auto reloading and an instant preview.
